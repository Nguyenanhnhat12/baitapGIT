Welcome to Our Flower Shop

This is a Node.js Pure Project (không dùng framework). Server sử dụng built-in modules: http, fs, path, url. Template giao diện Flower Shop được tích hợp sẵn.

Project bao gồm REST API quản lý Users với đầy đủ các chức năng: xem danh sách, xem chi tiết, thêm mới và xoá user. Dữ liệu được lưu trữ bằng file JSON.
